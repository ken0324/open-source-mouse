#すり減るマウス

使った分だけすり減るパーソナルなマウス．<br>
　−仕事量が見える．<br>
　−自分の手に馴染んでくる．<br>
　−愛着が湧いてくる．<br>
　−使いながら形が変わる／形を気にしながら使い方が変わる．<br>
　−自分のクセが見える．<br>
